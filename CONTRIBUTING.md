# Contributing to the Annie Mei docs

## How to contribute

### Edit on GitHub

1. Navigate to the page you want to edit
2. Click the pencil icon to edit the file
3. Submit a pull request

### Local development

1. Fork and clone this repository
2. Install the Mintlify CLI: `npm i -g mint`
3. Create a branch for your changes
4. Run `mint dev` from the repo root
5. Preview at `http://localhost:3000`
6. Submit a pull request

## Writing guidelines

Follow the style defined in [AGENTS.md](AGENTS.md):

- Use active voice and second person ("you")
- Keep sentences concise
- Use sentence case for headings
- Lead with the user outcome, then explain the technical detail
- Verify any implementation claims against the [source repo](https://github.com/annie-mei/annie-mei) before documenting them
